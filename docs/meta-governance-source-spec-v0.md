# Meta Governance Source Spec v0

작성일: 2026-04-04
상태: 초안

## 1. 목적

Agent Brain 자체의 설계, 규칙, 운영 원칙을 다루는 메타 운영 대화나 문서를 어떤 source 로 취급할지 정의한다.

## 2. 정의

Meta Governance Source 는 브레인 자체를 어떻게 설계하고 유지할지에 대한 결정을 담은 source 다.

예:
- Agent Brain 설계 스레드
- 운영 원칙 점검 대화
- 규칙/메모리/정체성 반영 결정
- channel/session ingest 정책 논의

## 3. 처리 원칙
- raw 에는 결정 맥락과 핵심 주제를 남긴다.
- compiled 에는 결정 요약과 다음 작업을 남긴다.
- 일반 리서치 source 와 구분해 상위 운영 source 로 취급한다.
- 다른 세션이 재사용해야 할 원칙은 MEMORY/IDENTITY/규칙 문서에도 반영한다.
