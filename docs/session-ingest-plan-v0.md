# Session Ingest Plan v0

작성일: 2026-04-04
상태: 계획 초안

## 목적

OpenClaw의 과거 세션 대화와 문서를 Agent Brain에 흡수해, 장기적으로 브레인 역할을 하게 하는 계획을 정리한다.

## 원칙
- 모든 세션을 한 번에 넣지 않는다.
- 우선 문서형 산출이 있는 세션부터 넣는다.
- 대화 전문보다는 raw 요약 + compiled summary/note 중심으로 시작한다.
- 반복되는 주제는 concept/index로 승격한다.
- 민감한 세션은 범위와 보존 수준을 따로 판단한다.

## 우선 대상
1. Agent Brain 관련 세션
2. 연구/오케스트레이션 관련 세션
3. OpenClaw 운영/아키텍처 관련 세션
4. Discord 연구 채널 관련 세션
5. Kakao 관련 세션

## 장기 방향
- Discord 열려 있는 채널을 하나씩 source로 흡수
- 필요 시 웹/문서 리서치로 missing data 보완
- 세션 -> raw -> compiled -> concept/index -> view 흐름으로 정착
